
### Cheat Sheet


| mode | command            | effect                          | link                                              |
|------|--------------------|---------------------------------|---------------------------------------------------|
| cmd  | `:split`           | splits vertically               |                                                   |
| cmd  | `^[hjkl]`          | window navigation               |                                                   |
| cmd  | ` t`               | open terminal window            |                                                   |
| cmd  | ` sf`              | find file by name               |                                                   |
| cmd  | ` sg`              | find file by grep               |                                                   |
| cmd  | `^v`               | MODE: visual block (VB)         |                                                   |
| cmd  | `:CopilotChatOpen` | MODE: llm chat (LLM)            | https://github.com/CopilotC-Nvim/CopilotChat.nvim |
| VB   | + `I` ... `<esc>`  | insert, apply, exit VB          |                                                   |
| LLM  | `^s`               | submit message in chat window   |                                                   |
| LLM  | `#file:src/a.txt`  | add a file to the conversation  |                                                   |
